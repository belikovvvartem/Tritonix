Політика конфіденційності TRITONIX STUDIO

TRITONIX STUDIO приділяє особливу увагу забезпеченню конфіденційності та захисту персональних даних користувачів. Ми прагнемо, щоб кожен клієнт розумів, як і для чого використовуються його дані, та був упевнений у їх безпеці.

1. Збір та використання персональних даних

TRITONIX STUDIO збирає та обробляє персональні дані, які надаються користувачами добровільно через форму замовлення або під час спілкування з представниками студії. До таких даних належать:

  • Ім’я та прізвище клієнта для персоналізації спілкування;
  • Контактні дані (номер телефону, електронна пошта) для оперативного зв’язку;
  • Дані про бажану послугу (опис, прикріплені файли).

Зібрані дані використовуються виключно для:

  • Виконання та обробки замовлень клієнта;
  • Зворотного зв’язку та консультацій;
  • Поліпшення наших послуг, включно з аналізом потреб клієнтів.

2. Права користувачів

Кожен користувач має право на:

  • Отримання доступу до своїх даних;
  • Виправлення даних;
  • Видалення даних;
  • Обмеження обробки;
  • Відкликання згоди.

Для реалізації будь-якого з цих прав необхідно звернутися до нас за контактами, зазначеними нижче.

3. Передача даних третім сторонам

TRITONIX STUDIO не передає ваші персональні дані третім сторонам, за винятком випадків, коли це необхідно для виконання зобов’язань перед клієнтом. Такі випадки включають:

  • Обробку платежів через платіжні системи;
  • Залучення кур’єрських або логістичних служб для доставки готового продукту.

4. Зберігання даних

Ваші персональні дані зберігаються на захищених серверах протягом часу, необхідного для виконання замовлення. Максимальний строк зберігання – 3 роки після завершення виконання замовлення, якщо інше не вимагається законодавством України.

5. Захист даних

TRITONIX STUDIO застосовує сучасні технічні й організаційні заходи для забезпечення безпеки персональних даних. Ми захищаємо дані від:

  • Несанкціонованого доступу чи розголошення;
  • Випадкового або навмисного знищення чи втрати;
  • Неавторизованої модифікації чи використання.

6. Файли cookie та аналітика

Для покращення користувацького досвіду наш вебсайт може використовувати файли cookie та інструменти аналітики. Ці дані не містять персональної інформації та використовуються лише для оптимізації роботи сайту.

7. Контактна інформація

У разі запитань або для реалізації ваших прав звертайтеся до нас:

  • Email: support@tritonixstudio.com

Ми завжди готові допомогти вам та надати повну інформацію про обробку ваших персональних даних.
